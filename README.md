# Course Management System - Udemy Module

A comprehensive course management system designed for the Udemy platform, focusing on video-based learning, automatic progress tracking, downloadable materials, and an interactive review system.

## üìö Project Overview

This project is part of **CSE 326: Information System Design Sessional** and implements a complete course management module for an EdTech platform. The system empowers instructors to create and manage courses while providing students with an engaging learning experience.

**Selected Platform:** Udemy (www.udemy.com)

## ‚ú® Core Features

### 1. üé• Video-based Lessons & Playback
- **Upload & Organization:** Instructors can upload video lectures in structured sections
- **Playback Controls:** Speed adjustment, quality selection, and subtitle support
- **Smart Resume:** System tracks viewing progress and enables resuming from last watched position
- **Accessibility:** Multi-format support with synchronized subtitles

### 2. üìä Automatic Progress Tracking
- **Auto-tracking:** Automatically marks lectures complete when viewing threshold is reached
- **Analytics Dashboard:** Detailed reports with completion percentages and milestones
- **Smart Notifications:** Automated reminders to boost course completion rates
- **Comprehensive Metrics:** Progress tracking for both students and instructors

### 3. üì• Downloadable Learning Materials
- **Rich Resources:** Support for PDFs, code files, slides, and more
- **Flexible Access:** Individual or bulk download options for offline learning
- **Version Control:** Resource versioning with download analytics
- **Section-specific Attachments:** Materials can be attached to specific course sections

### 4. ‚≠ê Course Rating & Review System
- **Authentic Reviews:** 1-5 star rating system with detailed written reviews
- **Two-way Communication:** Instructors can respond to student feedback
- **Quality Control:** Automated moderation system with admin oversight
- **Filtering Options:** Students can filter and sort reviews

## üë§ User Roles

### Student
- Browse and search courses
- Enroll in courses (free or paid)
- Watch video lectures with playback controls
- Download learning materials
- Track learning progress
- Write reviews and ratings
- Report inappropriate reviews

### Instructor
- Create and organize courses
- Upload video content and materials
- Monitor student progress
- View engagement metrics and analytics
- Reply to student reviews
- Manage course structure

### Admin
- Review moderation queue management
- Approve/reject reviews
- Handle violation reports
- System oversight and quality control
